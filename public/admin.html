    Twitch Chat aktivieren (optional)

    Damit Chat-Commands im Twitch-Chat funktionieren, brauchst du einen Bot-Token. Der Server laeuft auch ohne Twitch – teste Sounds ueber den Test Button.

    1. Bot-Token erstellen: twitchapps.com/tmi
    2. In .env eintragen:
      TWITCH_CHANNEL=dein_kanalname
      TWITCH_BOT_TOKEN=oauth:dein_token
    3. Server restarten (start.bat neu oeffnen)
